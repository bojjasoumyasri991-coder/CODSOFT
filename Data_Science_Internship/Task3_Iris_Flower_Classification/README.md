# Iris Flower Classification

## Objective
Classify iris flowers into three species using machine learning.

## Dataset
Built-in Iris dataset from scikit-learn.

## Steps
- Load dataset
- Train-test split
- Train Logistic Regression model
- Evaluate using accuracy and confusion matrix

## Result
Achieved accuracy greater than 95%.

## Tools Used
- Python
- Scikit-learn
- VS Code
